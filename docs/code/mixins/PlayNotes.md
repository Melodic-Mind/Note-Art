# PlayNotes

<a name="playNotes"></a>

## playNotes
Play a group of notes Harmonically.

**Kind**: global mixin  

| Param | Type | Description |
| --- | --- | --- |
| notes | <code>Array</code> | array of playable notes |
| duration | <code>string</code> | duration to play the notes for, e.g 4n. |
